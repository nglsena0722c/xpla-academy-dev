# 테스트넷 XPLA 토큰 받기

import VaultImgCard from "../../src/components/VaultImgCard";
import BrowserWindow from '@site/src/components/BrowserWindow';

XPLA 블록체인에서 송금도 하고 컨트랙트도 만들려면, XPLA 토큰으로 수수료를 지불해야 합니다. 그러나 개발이나 테스트를 위해 매번 수수료를 내기에는 비용 부담이 큽니다.

위와 같은 이유로 테스트넷이 존재합니다. 테스트넷에선 누구나 무료로 XPLA 토큰을 받아서 사용해볼 수 있습니다. 바로 시작해봅시다.

:::caution

테스트넷은 XPLA 블록체인의 발전을 위한 테스트 환경입니다. 테스트넷의 XPLA 토큰은 실제 가치가 없답니다. 따라서 상용화를 목적으로 테스트넷을 이용하는 것은 바람직하지 않습니다. 테스트 환경이므로 블록체인의 여러 조건이 바뀔 수도 있고, 심지어 없어질 수도 있기 때문입니다. 가령 이더리움의 Ropsten 테스트넷처럼 말이죠.

:::

:::tip

메인넷과 테스트넷은 이름(chainID)도 다릅니다! 메인넷의 chainID는 `dimension_37-1`이며, 테스트넷의 chainID는 `cube_47-5`입니다. 몇몇 분들은 [이전 단계](/docs/settings/create-wallet/create-wallet-with-js#runjs)에서 이미 눈치채셨을 수도 있겠습니다.

:::

## Faucet에서 XPLA 토큰 받기

테스트넷 XPLA 토큰을 받는 곳을 Faucet이라고 부릅니다. 다음과 같은 절차로 XPLA 토큰을 받을 수 있습니다.

1. [Faucet 사이트](https://faucet.xpla.io/)에 접속합니다.

2. 지갑 Address를 입력하고 `Send me tokens` 버튼을 눌러주세요.

3. Vault를 확인해보면 100 XPLA가 생겼다는 것을 확인할 수 있습니다. 만약 100 XPLA가 없다면, 우측 상단 톱니바퀴 아이콘을 눌러 네트워크를 testnet으로 바꿔보세요.
