# Explorer에서 트랜잭션 확인하기

import VaultImgCard from "@site/src/components/VaultImgCard";
import BrowserWindow from '@site/src/components/BrowserWindow';

[XPLA Explorer](https://explorer.xpla.io/)를 이용하면 트랜잭션 데이터를 자세히 살펴볼 수 있습니다. [이전 단계](/docs/settings/create-testnet-transaction/send-tx)에서 만든 트랜잭션 데이터를 다음과 같은 절차로 확인해봅시다.

1. [XPLA Explorer](https://explorer.xpla.io/)에 접속합니다.

<BrowserWindow url="https://explorer.xpla.io/" bodyStyle={{padding: 0}}>
<div className="flex justify-center" style={{backgroundColor : 'hsl(240, 20%, 97%)', position : 'relative'}}>
    <VaultImgCard style={{zIndex : 1, maxWidth : "100%"}} imgSrc="/xpla-academy-dev/img/2-settings/2-create-testnet-transaction/check-tx-in-explorer/go-explorer.png" />
</div>
</BrowserWindow>

<br/>

2. 네트워크를 `testnet`으로 바꿉니다.

<BrowserWindow url="https://explorer.xpla.io/" bodyStyle={{padding: 0}}>
<div className="flex justify-center" style={{backgroundColor : 'hsl(240, 20%, 97%)', position : 'relative'}}>
    <VaultImgCard style={{zIndex : 1, maxWidth : "100%"}} imgSrc="/xpla-academy-dev/img/2-settings/2-create-testnet-transaction/check-tx-in-explorer/select-testnet.png" />
</div>
</BrowserWindow>

<br/>

3. [이전 단계](/docs/settings/create-testnet-transaction/send-tx)에서 예제 코드를 실행했을 때, 결과로 얻은 해시값을 검색합니다. 만약 해시값을 기록하지 못했다면 아래 [다른 방법](/docs/settings/create-testnet-transaction/check-tx-in-explorer#checkwallet)으로 확인하실 수 있습니다.

<BrowserWindow url="https://explorer.xpla.io/testnet/" bodyStyle={{padding: 0}}>
<div className="flex justify-center" style={{backgroundColor : 'hsl(240, 20%, 97%)', position : 'relative'}}>
    <VaultImgCard style={{zIndex : 1, maxWidth : "100%"}} imgSrc="/xpla-academy-dev/img/2-settings/2-create-testnet-transaction/check-tx-in-explorer/search-tx.png" />
</div>
</BrowserWindow>

<br/>

4. 트랜잭션 데이터를 결과로 확인합니다.

<BrowserWindow url="https://explorer.xpla.io/testnet/tx/ECDBC35B66384DEE25987AA0DDAE8CF946D1F4907B04A0E77939988474013353" bodyStyle={{padding: 0}}>
<div className="flex justify-center" style={{backgroundColor : 'hsl(240, 20%, 97%)', position : 'relative'}}>
    <VaultImgCard style={{zIndex : 1, maxWidth : "100%"}} imgSrc="/xpla-academy-dev/img/2-settings/2-create-testnet-transaction/check-tx-in-explorer/see-tx.png" />
</div>
</BrowserWindow>

이처럼 [XPLA Explorer](https://explorer.xpla.io/)를 이용하면 트랜잭션 데이터에 대해 자세히 살펴볼 수 있습니다. 또, 어떤 트랜잭션이 실시간으로 생성되었는지, 현재 컨트랙트가 만들어졌는지 등 XPLA 블록체인에 대한 정보도 한눈에 살펴볼 수 있습니다.
