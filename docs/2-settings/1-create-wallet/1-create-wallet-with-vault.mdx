## 지갑 생성하기 {#createwallet}

Vault를 이용하면 다음과 같은 절차로, 지갑을 빠르게 만들 수 있습니다.

1. [New wallet 선택](/docs/settings/create-wallet/create-wallet-with-vault#clicknewwallet)

2. [지갑 정보 설정 및 니모닉(Mnemonic) 저장](/docs/settings/create-wallet/create-wallet-with-vault#savemnemonic)

3. [니모닉 검증](/docs/settings/create-wallet/create-wallet-with-vault#verifymnemonic)

4. [생성한 지갑을 Vault에 연결](/docs/settings/create-wallet/create-wallet-with-vault#connectwallet)

### New Wallet 선택 {#clicknewwallet}

Vault를 실행하고, `New wallet`을 선택하세요.

### 지갑 정보 설정 및 니모닉(Mnemonic) 저장 {#savemnemonic}

지갑에서 사용할 이름과 비밀번호를 기입하시고, 니모닉 단어들을 안전한 곳에 저장해주세요.

:::caution

니모닉은 지갑을 복구하기 위한 단어들입니다. 따라서 다른 사람에게 알려지거나, 도난되면 지갑의 모든 자금을 탈취당할 수 있습니다. 그러니 꼭 안전한 곳에 보관하시고, 어느 누구에게도 알려주면 안됩니다.

:::

이후 `Submit` 버튼을 클릭합니다.
