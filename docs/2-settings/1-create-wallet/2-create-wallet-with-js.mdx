### Node.js 설치 {#installnodejs}

여러분의 운영체제에 맞는 [Node.js](https://nodejs.org/en/download/)를 설치하세요.

Node.js가 잘 설치되었는지 확인하기 위해, Terminal을 열어 `node -v`를 입력해봅시다.

```bash
node -v
```

설치한 Node.js의 버전이 출력되면 설치가 완료된 것입니다.

Code 파일을 쉽게 편집하기 위해 Visual Studio Code를 이용하겠습니다. [Visual Studio Code](https://code.visualstudio.com/download)를 설치해주세요.

## Javascript 코드 실행하기 {#runjs}

이제 Quickstart에서 살펴본 코드를 직접 실행해봅시다.

1. [실행 환경 구성](/docs/settings/create-wallet/create-wallet-with-js#requirements)

2. [코드 실행](/docs/settings/create-wallet/create-wallet-with-js#runcode)

### 실행 환경 구성 {#requirements}

  1. 예제 코드 파일을 위한 폴더를 만들어주세요. 폴더 이름은 xpla-academy-example으로 부탁드립니다.

  2. 설치한 Visual Studio Code를 실행합니다.

  3. Visual Studio Code에서 File - Open Folder 를 통해 xpla-academy-example 폴더를 열어줍니다.

  4. 왼쪽 Explorer 창에서 폴더 이름 옆 아이콘(New file)을 클릭하여 `example-1.js` 파일을 생성합니다.
  <img src="/xpla-academy-dev/img/2-settings/1-create-wallet/vscode-newfile.png"  width="400" height="200" />

  5. 아래 코드를 복사하여 example-1.js 파일에 붙여넣고, 저장합니다.

  ```js
  const { MnemonicKey, LCDClient } = require("@xpla/xpla.js");

  const newKey = new MnemonicKey();
  console.log("mnemonic: " + newKey.mnemonic);

  const lcd = new LCDClient({
    chainID : 'cube_47-5', // xpla testnet network의 이름
    URL : 'https://cube-lcd.xpla.dev' // xpla testnet LCD URL
  });
  console.log("accAddress: " + lcd.wallet(newKey).key.accAddress);
  ```

### 코드 실행 {#runcode}

  1. Terminal - New Terminal 혹은 ctrl + j 단축키를 이용하여 VSCode에서 Terminal을 엽니다.

  2. 아래 명령어를 Terminal에 입력합니다.

  ```bash
  npm install @xpla/xpla.js
  node example-1.js
  ```

  3. Terminal에서 결과를 확인합니다. 니모닉 단어들과 주소를 확인할 수 있을 것입니다.
