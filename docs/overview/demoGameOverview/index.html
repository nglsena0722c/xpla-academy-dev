Demo Game Overview

XPLA ACADEMY는 Start Learning에서 학습하는 내용을 실제 Demo Project로 구현한 Break The Bricks 게임과 Web3 Gaming Ops.를 오픈소스로 제공합니다. 먼저 XPLA ACADEMY의 Web3 게임인 Break The Bricks의 구조를 살펴봅시다. Break The Bricks에는 START HERE 과정과 ADVANCED 과정이 있습니다. 먼저 START HERE 과정이 어떻게 구성되어 있는지 살펴봅시다.

START HERE은 게임 Tutorial로서 지갑 생성, 토큰 수령, NFT 민팅 등 Web3 기본 기능을 XPLA ACADEMY의 Start Leaning과 연결하여 학습할 수 있습니다. START HERE의 모든 스테이지를 완료하면, Break The Bricks 게임의 ADVANCED에서 사용하게 될 계정을 만들 수 있습니다. 이 계정은 START HERE의 STAGE 1에서 만든 지갑과 연결되어 데이터베이스에 저장됩니다.

START HERE 과정을 통해 게임 계정을 만들면, Break The Bricks 게임의 ADVANCED 과정을 즐길 수 있습니다. ADVANCED 과정이 이 어떻게 구성되어 있는지 살펴봅시다. ADVANCED 과정은 본격적인 게임 서비스이며, Web3 Gaming Ops.와 연결되어 토큰, NFT, 최고점수 블록체인 기록 등의 Web3 게임 운영을 체험할 수 있습니다.

Break The Bricks 게임 ADVANCED 과정을 통해 게임 재화인 DIAMOND()를 얻을 수 있습니다. Web3 Gaming Ops.에서는 DIAMOND()를 CW20 토큰인 ACADEMTY-TKN()으로 Convert할 수도 있고, ACADEMY-TKN()과 Testnet XPLA()를 Swap할 수도 있습니다. 유저는 NFT Shop에서 ACADEMY-TKN()을 지불하여 게임 장비인 PADDLE NFT를 민팅할 수 있고, PADDLE NFT를 게임 내에서 이용해서 더 높은 점수를 기록할 수 있습니다.

Web3 Gaming Ops.의 Record, Convert, Mint, Swap 4 가지 Web3 기능을 통해 게임과 블록체인을 연결해주었습니다. 여러분도 XPLA ACADEMY를 참고하여 직접 만든 게임과 XPLA 블록체인을 연결해보세요!
