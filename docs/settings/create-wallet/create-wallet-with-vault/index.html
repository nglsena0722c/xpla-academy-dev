Vault로 지갑 만들기

블록체인에서 지갑(wallet)은 유저의 토큰을 관리하는 소프트웨어를 의미하기도 하고, 유저의 계정(account) 자체를 의미하기도 합니다.

Vault 설치​

아래는 대표적인 지갑 앱들입니다. 앞으로 우리가 사용할 Vault Chrome Extension을 설치해봅시다.

Vault

XPLA 블록체인 네트워크에서 사용합니다.

Metamask

Ethereum 블록체인 네트워크에서 사용합니다.


Vault를 Chrome Extension으로 실행해보면 아래와 같은 화면을 볼 수 있습니다.

google.com

지갑 생성하기​

Vault를 이용하면 다음과 같은 절차로, 지갑을 빠르게 만들 수 있습니다.

  1. New wallet 선택

  2. 지갑 정보 설정 및 니모닉(Mnemonic) 저장

  3. 니모닉 검증

  4. 생성한 지갑을 Vault에 연결

New Wallet 선택​

Vault를 실행하고, New wallet을 선택하세요.

google.com

지갑 정보 설정 및 니모닉(Mnemonic) 저장​

지갑에서 사용할 이름과 비밀번호를 기입하시고, 니모닉 단어들을 안전한 곳에 저장해주세요.

caution

니모닉은 지갑을 복구하기 위한 단어들입니다. 따라서 다른 사람에게 알려지거나, 도난되면 지갑의 모든 자금을 탈취당할 수 있습니다. 그러니 꼭 안전한 곳에 보관하시고, 어느 누구에게도 알려주면 안됩니다.

이후 Submit 버튼을 클릭합니다.

chrome-extension://vault.xpla.io

니모닉 검증​

저장한 니모닉을 참고하여 화면에 나온 순서의 단어를 선택해주세요.

chrome-extension://vault.xpla.io

생성한 지갑을 Vault에 연결​

지갑 생성이 완료되었습니다. Connect를 선택하여 생성한 지갑을 Vault에 연결하면 끝입니다.

chrome-extension://vault.xpla.io

이제 Vault Chrome Extension을 실행하면 생성한 지갑 정보를 바로 확인할 수 있습니다.

google.com
