# Check Transactions (Explorer)

import VaultImgCard from "@site/src/components/VaultImgCard";
import BrowserWindow from '@site/src/components/BrowserWindow';

Using [XPLA Explorer](https://explorer.xpla.io/), you can explore transactions more in detail. Let's go through the following steps to check the transaction we created in the [previous section](/startlearning/settings/create-testnet-transaction/send-tx):

1. Access the [XPLA Explorer](https://explorer.xpla.io/).

<BrowserWindow url="https://explorer.xpla.io/" bodyStyle={{padding: 0}}>
<div className="flex justify-center" style={{backgroundColor : 'hsl(240, 20%, 97%)', position : 'relative'}}>
    <VaultImgCard style={{zIndex : 1, maxWidth : "100%"}} imgSrc="/img/2-settings/2-create-testnet-transaction/check-tx-in-explorer/go-explorer.png" />
</div>
</BrowserWindow>

<br/>

2. Change the network to `testnet`.

<BrowserWindow url="https://explorer.xpla.io/" bodyStyle={{padding: 0}}>
<div className="flex justify-center" style={{backgroundColor : 'hsl(240, 20%, 97%)', position : 'relative'}}>
    <VaultImgCard style={{zIndex : 1, maxWidth : "100%"}} imgSrc="/img/2-settings/2-create-testnet-transaction/check-tx-in-explorer/select-testnet.png" />
</div>
</BrowserWindow>

<br/>

3. Search for the hash value you obtained as a result when you executed the example code in the [previous step](/startlearning/settings/create-testnet-transaction/send-tx). If you didn't record the hash value, you can use an [alternative method below](/startlearning/settings/create-testnet-transaction/check-tx-in-explorer#checkwallet).

<BrowserWindow url="https://explorer.xpla.io/testnet/" bodyStyle={{padding: 0}}>
<div className="flex justify-center" style={{backgroundColor : 'hsl(240, 20%, 97%)', position : 'relative'}}>
    <VaultImgCard style={{zIndex : 1, maxWidth : "100%"}} imgSrc="/img/2-settings/2-create-testnet-transaction/check-tx-in-explorer/search-tx.png" />
</div>
</BrowserWindow>

<br/>

4. Find the transaction details as the result!
