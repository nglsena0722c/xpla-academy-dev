Demo Game Overview

✨Our demo projects, Break The Bricks and Web3 Gaming Ops, are open-source learning materials. Use them as you wish!

STARTHERE mode​

Break The Bricks has START HERE mode and ADVANCED mode. Let's look into the structure of START HERE mode.

Break The Bricks
Click "START HERE" in Game
START HERE
Create a Wallet
Wallet will be used on the Web3 Gaming Ops page, so save your mnemonic well.
Get Tokens
NFT Mint
Create Game ID

In START HERE mode, you experience followings :

✅ Wallet Creation
✅ Token Acquisition
✅ NFT minting
✅ Game Account <-> Web3 Wallets

All links connected to lessons provided on Start Learning.

🤫 Account <-> Wallet Spoiler In START HERE mode, users (you!) get to create a very simple game account! This account is linked to the wallet created in STAGE 1 of START HERE and is stored in the database.

Game accounts that's created through START HERE can open ADVANCED mode.

So, complete START HERE first! Create an account then try ADVANCED mode to fully interact with Web3 systems. Moreover, try applying it to your projects!

ADVANCED mode​

Now, let's look into the structure of ADVANCED mode.

Break The Bricks

Login with Game ID
ADVANCED
Select Paddle
Play Infinite Mode Game

Page in XPLA ACADEMY
Record your high score
Play Stage Mode Game
Get Diamonds

Mint a new paddle

and Testnet XPLA

ADVANCED mode has the core Web3 functions implemented :

✅ Connected to Web3 Gaming Ops.
✅ Using Tokens
✅ Using NFTs
✅ Make a Blockchain Records

All activities linked to Web3 Gaming Ops. In the Break The Bricks ADVANCED mode, you can earn the in-game currency DIAMOND(). In Web3 Gaming Ops, you can convert DIAMOND() to the CW20 token ACADEMY-TKN() and swap ACADEMY-TKN() with $XPLA() via testnet. Users can use ACADEMY-TKN() to buy the PADDLE NFT in the NFT Shop and use it in the game!

😎 The NFT Paddles don't really affect your gaming score, but they are helpful! Try achieving higher scorers with it.

Web3 Gaming Ops links the game and blockchain with four features:

🤟Record
🤟Convert
🤟Mint
🤟Swap
